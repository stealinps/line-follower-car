# line-follower-car
Line follower car made for cep final assignment 
Requirements:
- Motor driver control (L298N)
- WiFi mobile control interface
- Obstacle detection (ultrasonic)
- Line following (IR sensors)
- Optional camera module
- LED/Buzzer indicators

Sensors/Inputs:
-IR sensors
-Ultrasonic sensor
-Wifi
-Camera

Actuators/Output:
-LED
-Buzzer
-Motor
-Video
